# Land Area Converter

A modern React + Vite application for converting between Indian land measurement units with regional variations.

## Features

- Convert between multiple land measurement units
- Support for regional units across different Indian states
- Modern, responsive UI with Tailwind CSS
- Secure configuration with security headers
- Fast development experience with Vite

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```

## Security Features

- Content Security Policy (CSP) headers
- X-Frame-Options: DENY
- X-Content-Type-Options: nosniff
- X-XSS-Protection enabled
- Secure referrer policy
- Console logs removed in production build

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- ESLint for code quality
